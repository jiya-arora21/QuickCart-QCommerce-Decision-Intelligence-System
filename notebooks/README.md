# Notebooks

This folder contains Python notebooks used for data understanding, data cleaning, exploratory business analysis, KPI analysis, and decision intelligence development for the QuickCart project.

## Completed Notebooks

1. `01_data_understanding.ipynb`
   Initial dataset review, column understanding, data structure inspection, identifier checks, value distributions, and early business observations.

2. `02_data_cleaning.ipynb`
   Data cleaning, column standardization, time field handling, feedback grouping, business-ready feature creation, and cleaned dataset export.

3. `03_exploratory_analysis.ipynb`
   Exploratory business analysis covering platform performance, delivery performance, customer ratings, refund behavior, product categories, feedback categories, refund relationships, order value segments, customer frequency, platform-level repeat customer behavior, and EDA key findings.

## Planned Notebooks

4. `04_kpi_analysis.ipynb`
   Executive KPI analysis, KPI cards, refund KPIs, delivery KPIs, rating KPIs, customer KPIs, platform-level KPI summaries, and business performance indicators.

5. `05_decision_intelligence_recommendations.ipynb`
   Business-rule-based decision intelligence recommendations, risk indicators, operational improvement areas, customer experience actions, and final project recommendations.
